# IdeaVault - Startup Idea Sharing Platform

**Live URL:** [https://ideavault.vercel.app](https://ideavault.vercel.app)

## Features

- Modern Next.js 14 + App Router
- JWT Authentication with Email & Google Login
- Idea sharing with category, tags, budget, and problem-solution
- Comment, Edit & Delete comments
- Search & Filter (title + category + date range)
- Dark/Light theme toggle
- Fully responsive (Mobile, Tablet, Desktop)
- Protected routes with PrivateRoute
- Toast notifications for all actions

## Tech Stack

- Next.js 14 (App Router)
- Tailwind CSS
- Firebase Authentication
- React Hot Toast
- Lucide Icons
